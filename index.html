Welcome to the GitHub repository for the Component Contribution method

This repository hosts the most up-to-date version of component contribution for Python and Matlab. Please treat this code as alpha, since no real versioning is being done. If you wish to have a clean and simple interface with stable Gibbs energy estimations, please try eQuilibrator.
